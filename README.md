# 项目简介

## 核心功能
- 学生、教师、管理员登录
- 前后端代码及SQL文件
- 万字文档
- 参考导入调试视频、讲解视频、开发工具安装包

## 适用场景
- 教学管理
- 学生信息管理
- 教师信息管理
- 管理员权限管理

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
